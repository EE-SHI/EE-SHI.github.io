晚秋登高

夜游登北坡，南望见星河。
云为西风散，愁来晚秋多。
十年读寂寞，万里作漂泊。
灯火阑珊处，早行客几多。

中秋咏怀

发表于 2021-10-02 更新于 2021-11-12 分类于 随笔

   中秋羁旅斯德哥尔摩，原定于一个月前回国却因疫情无奈推迟。举目无亲，出门与朋友相聚又恐染疫更加耽误行程，焦虑烦躁种种心思不一而足，作绝句两首以为消遣排解。

其一

秋风秋月秋思量，岁华摇落牵愁肠。
天涯多少羁行客，尽望玉轮作故乡。

其二

枯荣总有时，圆缺亦无情。
此物有定数，奈何人伤魂。

少年时的情诗

发表于 2021-09-26 更新于 2021-09-27 分类于 随笔

   大学深夜无眠之时油然想起追求过的第一个姑娘，我们初中相识，彼此暧昧却又太过害羞。初三上半我便知道毕业后会去西安读高中，而她则确定留在宝鸡。终于鼓起勇气写了一封告白信，想着能谈一时恋爱便是一时，换得一封“拒信”，信中“无力打搅”四字记忆尤深。遂有此小诗。
   “暧昧是爱情最美也最伤人的部分。”

名字和记忆升腾
你说无力打搅
那时的歌、信和暧昧
一一衰老

无力打搅
一切可好

废都

发表于 2021-09-26 更新于 2021-09-27 分类于 随笔

废都

高中时的意气之作，少年心中的长安梦。

那该死的唤你废都
我想踹他们一脚
然后自打着灯笼
从钟鼓楼下奔到朱雀街上
只为杀出一个盛唐
长安长安
凭吊盛唐
书院门里不见诗酒
雁塔寺下独剩经房
噫嘘废矣
胡为而来哉
少年心中有个盛唐

我打着灯笼
从未央走到朱雀背上
从朱雀走到太白身旁
杀出一个盛唐
杀出一个盛唐
